This section introduces two important mathematical data structures.

Sets

A set is a collection of objects that has no duplicates, and is not in any particular order. The most common notation for a set is to use curly braces. For example, $\{0, 1, 2, 3, 4\}$ is a set of five numbers. Because sets are unordered, $\{0, 1, 2, 3, 4\}$ is the same as $\{4, 1, 3, 0, 2\}$. And, because sets don't have duplicates, $\{0, 0, 1, 2, 3, 4, 4\}$ equals $\{0, 1, 2, 3, 4\}$. We say two sets are equal if they contain exactly the same elements.

Set notation

Sets can be described in terms of some predicate. The set of all things such that $p(x)$ is true is written as $\{x \mid p(x)\}$, or sometimes $\{x : p(x)\}$. For example, the set $E$ of all even integers is $E = \{2x \mid x \mbox{ is an integer}\}$. You would read this as "$2x$ for all $x$ such that $x$ is an integer."

If we wanted to say that a number $e$ is in the set of even numbers $E$, we would write $e \in E$. For example, the set of odd numbers is $\{e + 1 \mid e \in E\}$. To say an element is not in a set, we use $\notin$, as in $3 \notin E$.

A set that has nothing in it is called the empty set, denoted with the symbol $\emptyset$.

The number of elements in a set is denoted as $\left\vert A\right\vert$. For example, $\left\vert\{x \mid x \mbox{ is even} \land x \geq 0 \land x < 10\}\right\vert = 5$, and $\left\vert\emptyset\right\vert = 0$.

Set operations

Let's say we have two sets, $A$ and $B$. The set of all elements that are either in $A$ or $B$ is called the union of $A$ and $B$, denoted $A \cup B$. Formally, $A \cup B = \{e \mid (e \in A) \mbox{ or } (e \in B)\}$.

The intersection of $A$ and $B$ (denoted $A \cap B$) is the set of elements that are in both $A$ and $B$. That is, $A \cap B = \{e \mid (e \in A) \mbox{ and } (e \in B)\}$.

The set of all elements in $A$ that are not in $B$ is $A \setminus B$, called "set minus". This is sometimes also written as $A - B$. You can think of it as subtracting out all the elements of $B$ from $A$. Formally, $A \setminus B = \{e \mid e \in A \mbox{ and } e \notin B\}$.

The complement of $A$ (denoted $A^C$) is the set of all elements that are not in $A$. In other words, $A^C = \{e \mid e \notin A\}$. We assume that there is some universe of possible elements, $U$, such that $A^C = U \setminus A$.

Subsets

If all the elements of $A$ are also in $B$, then $A$ is a subset of $B$, denoted as $A \subseteq B$. In that case, $B$ is called a superset of $A$. For example, the set of even numbers is a subset of the set of integers. The empty set is a subset of all sets. It has no elements, so trivially, "all" of its elements are in any other set.

Every set is a subset of itself. If we want to say that $A$ is a subset of $B$ and $A$ is smaller than $B$, then we say that $A$ is a strict subset of $B$, denoted $A \subset B$.

Power set

The power set of $A$ is $P(A) = \{X \mid X \subseteq A\}$, the set of all subsets of $A$. For example, the power set of $\{1, 2, 3\}$ is:

$$\left\{\emptyset, \{1\}, \{2\}, \{3\}, \{1, 2\}, \{1, 3\}, \{2, 3\}, \{1, 2, 3\}\right\}$$

To be clear, the power set is a set whose elements are also sets. As you can see, the power set of $\{1, 2, 3\}$ has 1 empty set, 3 sets of size one, 3 sets of size two, and 1 set of size three, for a total of 8 elements.

Set identities

As with boolean logic, there are identities that apply to sets. We will see how to prove these in the next section. Some important identities include:

Associativity
$A \cup (B \cup C) = (A \cup B) \cup C$
$A \cap (B \cap C) = (A \cap B) \cap C$
Distributivity
$A \cup (B \cap C) = (A \cup B) \cap (A \cup C)$
$A \cap (B \cup C) = (A \cap B) \cup (A \cap C)$
Idempotency
$A \cup A = A$
$A \cap A = A$
Double complement
$(A^C)^C = A$
DeMorgan's laws
$(A \cup B)^C = A^C \cap B^C$
$(A \cap B)^C = A^C \cup B^C$
Commutativity
$A \cup B = B \cup A$
$A \cap B = B \cap A$
Absorption
$A \cup (A \cap B) = A$
$A \cap (A \cup B) = A$

Proving set identities

One way to prove that $A$ and $B$ are equal is to show that $A$ is a subset of $B$, and $B$ is a subset of $A$. Because $A$ is a subset of $B$, $B$ contains all the elements that $B$ contains. So as long as $B$ doesn't have any elements that $A$ doesn't have, $A$ and $B$ contain exactly the same elements. We know this is true, however, because $B$ is a subset of $A$, which, by definition, means that any element in $B$ is also in $A$.

Let's prove the first of DeMorgan's laws, $(A \cup B)^C = A^C \cap B^C$, by showing that each side of the equation is a subset of the other side.

Proof:

First we show that $(A \cup B)^C \subseteq A^C \cap B^C$:

  1. Let $x$ be any element in $(A \cup B)^C$.
  2. By definition of set complement, $x \notin (A \cup B)$.
  3. $x \notin A$, since if $x$ were in $A$, $x$ would be in $A \cup B$ as well. Therefore, by definition of set complement, $x \in A^C$.
  4. $x \in B^C$ for the same reason.
  5. Because $x \in A^C$ and $x \in B^C$, $x \in A^C \cap B^C$ by definition of set intersection.
  6. Since any element of $(A \cup B)^C$ is also in $A^C \cap B^C$, $(A \cup B)^C$ is a subset of $A^C \cap B^C$ by definition of subsets.

Next we show that $A^C \cap B^C \subseteq (A \cup B)^C$:

  1. Let $x$ be any element in $A^C \cap B^C$.
  2. By definition of set intersection, $x \in A^C$ and $x \in B^C$.
  3. $x \notin A$ and $x \notin B$ by definition of set complement.
  4. $x \notin (A \cup B)$. Suppose instead that $x \in (A \cup B)$. Then $x \in A$ or $x \in B$ by definition of set union. Let $p(x) = ``x \in A\mbox{ or }x \in B."$ By DeMorgan's law for boolean logic, $\neg p(x) = ``x \notin A\mbox{ and }x \notin B."$ But we showed earlier that this was true. So $p(x)$ is true and $\neg p(x)$ is true, which is a contradiction. Therefore, our assumption that $x \in (A \cup B)$ was false, and $x \notin (A \cup B)$.
  5. By definition of set complement, $x \in (A \cup B)^C$.
  6. Since every element of $A^C \cap B^C$ is in $(A \cup B)^C$, $A^C \cap B^C$ is a subset of $(A \cup B)^C$ by definition of subsets.

We have shown that $(A \cup B)^C \subseteq A^C \cap B^C$ and $A^C \cap B^C \subseteq (A \cup B)^C$, therefore $(A \cup B)^C = A^C \cap B^C$. $\square$

The above proof shows an example of proof by contradiction. This is just one of many proof techniques that are discussed more in other notes. As you can see, these kinds of proofs can be very tedious. Luckily, once someone proves something once, it's proven forever.

Common sets of numbers

Below are common symbols used for sets of numbers.

SymbolNumber set
$\mathbb{N}$All natural numbers
$\mathbb{Z}$All integers
$\mathbb{Z}_+$All positive numbers
$\mathbb{Z}_n$The set $\{0, 1, \ldots, n-1\}$
$\mathbb{Q}$All rational numbers
$\mathbb{R}$All real numbers

The $\mathbb{Z}$ for integers comes from the German word "zahlen," meaning "numbers." $\mathbb{Q}$ comes from "quotient."

Tuples

An $n$-tuple is an ordered collection of $n$ elements. The size of a tuple is finite and fixed -- you can't add an element to it or remove an element from it. Unlike sets, they can contain duplicate elements. They are commonly used to group different pieces of information together. For example, you might say that a person's birthday is a 3-tuple, where the first element is the year, the second element is the month, and the third element is the day. Tuples are commonly written using parentheses, like $(y, m, d) = (1990, 5, 1)$.

Cartesian product

The Cartesian product of two sets is $A \times B = \{(a, b) \mid a \in A\mbox{ and }b \in B\}$. For example, $\mathbb{R} \times \mathbb{R}$, the set of all pairs of real numbers, is the $xy$-plane we're all familiar with from algebra. $\mathbb{R} \times \mathbb{R}$ is often denoted $\mathbb{R}^2$. The Cartesian product can be generalized to handle any number of sets. For example, $A \times B \times C = \{(a, b, c) \mid a \in A\mbox{ and }b \in B\mbox{ and }c \in C\}$. The set of all 3D points is the Cartesian product $\mathbb{R}^3$.

Other structures

What about other containers, such as an ordered set, or a set that can contain more than one element? A set that can contain duplicates is called a multiset. You may encounter it at some point while studying computer science, but it's not common.

Order is typically imposed on sets by adding indices to each element. For example, let $X$ be a set of $n$ students in a class ($\left\vert X \right\vert = n$). Then let $x_1, x_2, \ldots, x_n \in X$ be students such that $x_i < x_{i+1}$ if $x_i$ is shorter than $x_{i+1}$. This orders the students by increasing height. Or, we could say $x_i < x_{i+1}$ if $x_i$ has a better grade than $x_{i+1}$. Then, we would be ordering the students by decreasing grade.
